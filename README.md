# FormProcess

Este é um projeto simples de aplicação web para processamento de formulários, construído com Node.js, Express e TypeScript. Ele utiliza Handlebars para a renderização das visualizações no lado do servidor.

## Funcionalidades

- Exibe uma página inicial.
- Apresenta um formulário para entrada de dados do usuário.
- Processa os dados do formulário e exibe uma página de conclusão.

## Como Executar o Projeto

1.  **Instale as dependências:**
    ```bash
    npm install
    ```

2.  **Execute em modo de desenvolvimento (com recarregamento automático):**
    ```bash
    npm run dev
    ```

3.  **Execute em modo de produção:**
    ```bash
    npm start
    ```

4.  **Para fazer o build do projeto:**
    ```bash
    npm run build
    ```
